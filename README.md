## Fundamental Booster
## Objective
Create an **Interactive Personal Data Collector** application in Python that captures, processes, and displays personal information from the user using fundamental Python functions and concepts. 

This project demonstrates an understanding of:
* `print()` and `input()` functions
* Data types & Variables
* Arithmetic operators
* Type casting
* Built-in functions like `id()` and `type()`

## 📋 Requirements & Features

### 1. Detailed Use of `print()` and `input()` Functions
* **Gather Data:** Uses `input()` to collect personal information such as name, age, height, and favorite number.
* **User Guidance:** Uses `print()` to display formatted information and step-by-step guidance.

### 2. Data Types, Variables & Operators
* **Storage:** Stores each piece of collected information in an appropriately typed variable.
* **Calculations:** Demonstrates arithmetic operators (e.g., `+`, `-`, `*`, `/`) to perform calculations, such as estimating birth year from age.
* **Formatting:** Employs string concatenation or f-strings to build user-friendly outputs.

### 3. Type Casting Constructors
* **Explicit Casting:** Ensures data is properly cast (e.g., age to `int`, height to `float`).
* **Conversions:** Converts types where necessary (e.g., converting float to int for rounded values) and explains the conversion clearly in the terminal.

### 4. `id()` and `type()` Functions
* **Inspection:** Displays the precise data type and memory address of each variable using `type()` and `id()`.
* **Summary Table:** Outputs a clear summary of each variable showing its name, value, data type, and memory location.
